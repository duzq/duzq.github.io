gitosis配置

Git 1.8.2.2 + gitosis

1.查看Linux系统服务器系统版本

2.在服务器上安装git及做些操作

  • 执行命令
    sudo yum install curl-devel expat-devel gettext-devel openssl-devel zlib-devel perl-devel
  • 同时下载git-1.8.2.2.tar.gz文件，然后将其mv 到/usr/local/src
  • 增加软连接

3.在服务器安装gitosis

4.在开发机上,生产密钥并上传到服务器上

或编辑/etc/hosts文件，在/etc/hosts文件里添加如下文本：

然后再上传自己的公钥到服务器

5.服务器上生成git用户，使用git用户并初始化gitosis

6.在个人开发机上导出项目管理

7.在个人开发机增加及设置管理项目

8.在个人开发机上初始，增加及使用项目test-git

9.增加协同开发者的公钥key到git服务器

  • 执行cd repo/gitosis-admin/keydir切换目录

  • 把协同开发者的id_rsa.pub 文件里的数据 拷贝到 对应的开发者的密钥用户名.pub文件。如把密钥用户名 guangyun.ni@yeepay.com 的 id_rsa.pub 文件中文本 粘贴到 guangyun.ni@yeepay.com.pub 文件里，并保存

  • 然后将添加数据后的目录更新到git服务器
